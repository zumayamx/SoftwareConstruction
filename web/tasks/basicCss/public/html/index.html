Font
La propiedad font permite establecer de una sola vez los valores para todas las propiedades: font-style, font-variant, font-weight, font-size, line-height y font-family
Ejemplo de font
Font Family
La propiedad font-family define una lista de fuentes o familias de fuentes, con un orden de prioridad, para utilizar en un elemento seleccionado. Separando por comas los valores alternativos, es una elección de fuentes en caso de que la primera no este disponible para el usuario
Ejemplo de font-family

Margin
La propiedad CSS margin establece el margen para los cuatro lados. Es una abreviación para evitar tener que establecer cada lado por separado con las otras propiedades de margen: margin-top, margin-right, margin-bottom y margin-left.
Ejemplo de margin, observe como toma valor de 100px definido en el styles.css
Padding
La propiedad abreviada de CSS padding establece el área de relleno en los cuatro lados de un elemento a la vez.
Ejemplo de padding, observe como toma valor de 50px definido en el styles.css
